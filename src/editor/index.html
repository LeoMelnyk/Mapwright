Release Notes
AI Settings

Mapwright uses a local Ollama instance for AI dungeon generation — no API key or internet connection required.

  1. Download from ollama.com and install
  2. Run: ollama pull qwen2.5:14b
  3. Ollama starts automatically in the background
Runs locally — free, private, no usage limits.
The model must fit in your GPU VRAM — choosing one that's too large will cause errors or extremely slow responses. The GB shown is approximate VRAM required at 4-bit quantization.
Enable AI Agent?

The AI Agent feature is experimental. It uses a local Ollama model to build and modify dungeons from natural language.

Make sure Ollama is running and a model is pulled before using it. Configure the model in Help → AI Settings.
